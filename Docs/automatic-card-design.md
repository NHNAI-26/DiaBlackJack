> 최종 갱신: 2026-07-25

## 1. 문서 목적

수동 카드 사용과 악마 계약에 이어, 공개 영역에 들어온 카드가 별도 행동을 소비하지 않고 즉시 효과를 일으키는 **자동 발동 카드 시스템**을 독립 작업으로 정의한다.

이 문서는 다음을 확정한다.

- 자동 발동을 검사하는 카드 유입 시점
- 여러 효과와 버스트 사이의 처리 순서
- 효과 선택 중 전투를 멈추고 재개하는 방식
- 효과 처리 후 원본 카드의 위치
- 플레이어와 적에게 동일하게 적용되는 카드별 규칙
- 숨은 정보와 적 AI의 공정성 경계
- 구현 전에 기획자가 임시로 결정해야 하는 충돌 사항

구현하지 않은 기능은 완료로 기록하지 않는다. 이천서는 기획·개발·검증의 최종 책임자이며 AI는 규칙 대조, 구조 초안과 검증 보조로만 기록한다.

## 2. 현재 기준선

2026-07-23 AC-00 작성 시점의 실제 상태는 다음과 같다.

- `CardActivationKind.Automatic` 분류는 존재하지만 자동 발동 카드 정의와 실행기는 없다.
- 수동 카드 4종과 사탄의 권능은 `CardEffectResolver` 및 타입별 처리기로 실행된다.
- 플레이어와 적의 수동 카드 선택, 진행 세션 전달, 전투 UI가 구현돼 있다.
- 최초 배분, 일반 히트, 강제 히트, 수정 구슬 획득은 서로 다른 코드 경로로 카드를 손에 넣는다.
- 중간 버스트는 공개 합, 양쪽 스탠드 뒤 최종 승부는 비공개 카드를 포함한 전체 합으로 판정한다.
- 적 AI는 공개 관측만 사용하며, 카드 효과가 합법적으로 제공한 정보만 별도 결과로 받을 수 있다.
- 직전 자동 검증 기준은 DC-08 전체 EditMode 397/397이지만 AC-00에서는 Unity를 다시 실행하지 않는다.

## 3. 목표와 제외 범위

### 3.1 이번 시스템의 목표

- 공개 영역 카드 유입을 한 경계에서 감지한다.
- 자동 카드 효과가 끝나기 전에는 버스트·다음 행동·외부 효과를 진행하지 않는다.
- 플레이어 선택이 필요한 효과는 입력을 기다리고, 적 선택은 결정적인 정책으로 해결한다.
- 자동 효과가 끝난 뒤 원래 히트·카드 효과·적 행동을 정확히 한 번 재개한다.
- 독극물, 부활초, 거짓말 탐지기, 화염 방사기, 회중시계 5종을 양측 대칭으로 구현한다.
- 자동 카드가 보상과 런 덱을 거쳐 실제 전투에 등장하게 한다.

### 3.2 이번 시스템에서 제외하는 항목

- 나머지 악마 카드와 바포메트 오망성
- 카드 강화안과 유물
- 상점, 골드와 정식 런 순서
- 신규 애니메이션·사운드·외부 에셋
- 최초 배분 카드의 자동 발동
- 자동 카드가 비공개 상태로 손에 들어왔을 때의 발동

상점과 정식 런 진행은 HONG의 계획 영역을 유지한다. 이천서는 전투·카드·적 AI·현재 전투 UI 경계만 담당한다.

## 4. 공통 발동 규칙

### 4.1 발동 조건

자동 카드는 다음 조건을 모두 만족할 때 발동한다.

1. 카드 정의의 발동 종류가 `Automatic`이다.
2. 최초 라운드 배분이 끝난 뒤 카드가 새로 공개 영역에 추가된다.
3. 같은 공개 영역 진입에 대해 아직 자동 효과를 처리하지 않았다.

다음 유입은 발동 대상이다.

- 플레이어 또는 적의 일반 히트
- 보위 나이프·사탄의 권능 등 강제 공개 드로우
- 수정 구슬 등 카드 효과로 가져온 공개 카드
- 향후 효과가 공개 영역에 직접 생성하거나 이동하는 카드

다음 유입은 발동 대상이 아니다.

- 라운드 시작의 최초 공개·비공개 카드 배분
- 체인지로 선택한 새로운 비공개 카드
- 위협용 해머가 교체한 비공개 카드
- 이미 공개 영역에 남은 카드의 상태 변경
- 회중시계가 사용 완료 카드를 다시 사용 가능하게 만드는 동작

### 4.2 처리 순서

공개 카드 한 장의 유입은 아래 순서를 지킨다.

```text
카드 공개 영역 진입
  → 자동 카드 여부 확인
  → 자동 효과와 모든 선택 완료
  → 원본 카드 위치 처리
  → 현재 공개 합 버스트 확인
  → 중단했던 히트·카드 효과·적 행동 재개
```

- 자동 효과 처리 중에는 일반 행동, 수동 카드 사용, 체인지와 계약을 모두 잠근다.
- 효과가 영혼 0, 버스트 또는 라운드 초기화를 확정하면 남은 자동 발동과 원래 행동의 후속 처리를 취소한다.
- 여러 카드가 한 명령으로 공개 영역에 추가되면 실제 추가 순서대로 한 장씩 처리한다.
- 한 카드의 자동 효과와 버스트 검사를 마치기 전에는 다음 카드의 효과를 시작하지 않는다.
- 자동 효과는 별도 차례 행동으로 세지 않는다. 자동 카드를 뽑게 한 원래 행동만 차례를 소비한다.

### 4.3 최초 배분 예외

최초 배분으로 받은 자동 카드는 발동하지 않는다.

- 최초 공개 카드는 공개 영역에 남아 숫자를 합계에 포함한다.
- 최초 비공개 카드는 비공개 상태로 남아 최종 승부에만 숫자를 포함한다.
- 같은 카드가 이후 버린 더미를 거쳐 다시 공개 영역으로 들어오면 정상 발동한다.

이 예외는 시작하자마자 선택 창이 연쇄적으로 열리는 문제와 비공개 카드 수 불변식 훼손을 방지한다.

## 5. 카드 위치와 수명

### 5.1 기본 위치 규칙

- 자동 카드 원본은 효과 해결이 끝난 뒤 기본적으로 소유자의 버린 카드 더미로 이동한다.
- 원본 카드를 버린 뒤 공개 합을 계산하므로 기본 폐기 카드의 숫자는 발동 후 버스트 판정에 남지 않는다.
- 효과가 라운드나 전투를 즉시 끝내더라도 카드 총수와 물리 ID가 중복되거나 유실되지 않아야 한다.
- 자동 효과가 보류된 동안 원본은 공개 손패에 남지만 다른 효과의 선택 대상이나 수동 사용 대상이 될 수 없다.

### 5.2 회중시계 예외

회중시계는 효과 처리 뒤 소유자가 `즉시 버림` 또는 `공개 영역에 유지`를 선택한다.

- 유지한 회중시계는 숫자 9로 공개 합에 남는다.
- 같은 공개 영역에 머무는 동안 다시 발동하지 않는다.
- 이후 버린 뒤 다시 공개 영역에 들어오면 새로운 발동으로 처리한다.
- 회중시계 자신과 다른 자동 카드는 재사용 가능 대상으로 선택할 수 없다.

## 6. 카드별 프로토타입 규칙

### 6.1 독극물 — 숫자 2

발동 소유자는 다음 중 하나를 선택한다.

1. 지금 즉시 스탠드한다.
2. 영혼 3개를 잃는다. 현재 영혼이 3보다 적으면 가진 영혼을 전부 잃는다.

프로토타입 결정은 다음과 같다.

- 활성 계약이 스탠드를 금지하면 계약 제한이 우선하며 `즉시 스탠드` 선택지는 제공하지 않는다.
- 영혼 지불은 계약 시작 비용과 달리 최소 1을 보존하지 않는다. 영혼 0이면 즉시 전투 패배다.
- 영혼을 지불하고 현재 라운드에서 승리하면 영혼 5를 회복한다. 최대 영혼을 넘지 않는다.
- 부활초로 승패 없이 라운드를 초기화하면 독극물 회복은 발생하지 않는다.
- 같은 라운드에서 여러 독극물 비용을 지불하면 물리 카드별 회복 예약을 각각 보존한다.
- 원본 독극물은 효과 해결 후 버린다.

### 6.2 부활초 — 숫자 2

발동 소유자는 `새 라운드 시작` 또는 `사용하지 않음`을 선택한다.

새 라운드 시작을 선택하면 다음을 원자적으로 처리한다.

1. 양측이 영혼 1씩을 동시에 잃는다.
2. 승자와 일반 라운드 피해 없이 현재 양측 손패를 모두 버린다.
3. 현재 라운드 전용 효과와 보류 중인 외부 효과를 정리한다.
4. 다음 라운드를 최초 배분 규칙으로 시작한다.

프로토타입에서는 양측 모두 영혼이 2 이상일 때만 새 라운드 선택을 허용한다. 어느 한쪽이라도 영혼 1이면 해당 선택을 숨기고 `사용하지 않음`만 제공한다. 이는 양측 영혼 0의 동시 사망 우선순위가 현재 전투 결과 모델에 없고, 카드 문구의 목적이 사망 교환보다 라운드 재시작에 있기 때문이다.

- 부활초 초기화는 라운드 승패가 아니므로 일반 피해, 정확히 21 보너스와 독극물 회복을 발생시키지 않는다.
- 계약의 라운드 종료 훅은 한 번 호출하고 라운드 전용 상태는 초기화한다.
- 부활초가 다른 카드 효과의 강제 드로우로 발동했다면 바깥 효과와 남은 차례 처리를 취소한다.
- 원본 부활초는 사용 여부와 관계없이 효과 해결 후 버린다.

### 6.3 거짓말 탐지기 — 숫자 3

발동 소유자는 1부터 10까지 숫자 하나를 선언한다.

- 상대의 비공개 카드 숫자가 선언 값 이상이면 `이상`, 작으면 `미만`으로 응답한다.
- 선언 숫자는 공개 정보다.
- `이상/미만` 응답은 효과 소유자만 확인한다.
- 결과와 공용 전투 로그에는 실제 비공개 숫자를 저장하지 않는다.
- 적이 사용한 경우 적의 합법적인 전투 지식에는 비교 결과만 저장하고 실제 숫자는 전달하지 않는다.
- 상대 비공개 카드가 정확히 한 장이 아니면 `판정 불가`로 안전하게 종료하고 숨은 값을 읽지 않는다.
- 체인지, 해머 교체 또는 새 라운드로 상대 비공개 카드가 바뀌면 이전 비교 지식을 폐기한다.
- 원본 거짓말 탐지기는 효과 해결 후 버린다.

### 6.4 화염 방사기 — 숫자 9

발동 시점에 스탠드하지 않은 각 참가자는 자신의 공개 카드 한 장을 버리거나 건너뛸 수 있다.

- 선택 순서는 효과 소유자, 상대 순이다.
- 스탠드한 참가자는 선택 단계 자체를 건너뛴다.
- 화염 방사기 원본은 기본 폐기 예정이므로 버릴 카드 후보에서 제외한다.
- 자신의 다른 공개 카드가 없으면 자동으로 건너뛴다.
- 먼저 선택한 참가자의 카드 이동을 반영한 뒤 다음 참가자의 선택을 처리한다.
- 양측 선택과 원본 폐기를 모두 끝낸 뒤 공개 합 버스트를 검사한다.
- 원본 화염 방사기는 효과 해결 후 버린다.

적은 자신의 공개 합이 17 이상이거나 이미 버스트 위험이면 숫자가 가장 큰 유효 카드를 버리고, 그 외에는 기본적으로 건너뛴다. 이 수치는 최초 프로토타입 정책이며 플레이 테스트로 조정한다.

### 6.5 회중시계 — 숫자 9

발동 소유자는 자신의 공개 영역에 있는 사용 완료 수동 카드 한 장을 다시 사용 가능한 상태로 만들 수 있다.

- 대상은 `Manual`이면서 `Used`인 자신의 공개 카드다.
- 사탄의 권능도 현재 면이 사용 완료 상태라면 대상이 될 수 있다.
- 회중시계 자신, 다른 자동 카드와 이미 사용 가능한 카드는 대상이 아니다.
- 유효 대상이 없으면 재사용 단계는 자동으로 건너뛴다.
- 재사용 가능 상태로 바꾸는 것은 효과 실행이 아니므로 자동 연쇄를 만들지 않는다.
- 대상 처리 뒤 회중시계를 즉시 버릴지 공개 영역에 유지할지 선택한다.
- 유지한 회중시계는 해당 손패를 떠나기 전까지 다시 발동하지 않는다.

### 6.6 AC-04 구현 확정

- 화염 방사기의 두 선택은 하나의 동시 선택이 아니라 소유자와 상대의 연속 상호작용으로 구현한다.
- 첫 폐기 결과를 손패와 버린 더미에 반영한 뒤 상대 후보를 새로 계산한다.
- 화염 방사기 원본은 소유자 후보에서 제외하며 두 단계 완료 뒤 공통 자동 카드 완료 경계에서 버린다.
- 회중시계의 재활성화는 카드 효과 실행이 아니라 `Used → Available` 상태 변경만 수행한다.
- 회중시계 원본의 유지·폐기는 재활성화 대상과 분리된 두 번째 선택이며, 유지 시 자동 카드의 `Unavailable` 상태를 보존한다.
- AC-04에서는 사람·적 선택 규칙만 대칭으로 만들고 적의 자동 우선순위와 화면 입력은 AC-06에 남긴다.

### 6.7 AC-05 구현 확정

- 부활초의 `라운드 재시작`은 양측 현재 영혼이 모두 2 이상일 때만 선택할 수 있다.
- 재시작하면 부활초 원본을 버리고 양측 영혼을 각각 1 감소시킨 뒤, 승자·일반 피해·정확히 21 보너스·독극물 회복 없이 다음 라운드를 시작한다.
- 일반 `RoundResolution`을 위조하지 않고 `RoundTransition`에 원인, 이전·새 라운드 번호, 원본 물리 카드 ID와 소유자를 기록한다.
- 재시작은 독극물 승리 회복 예약, 거짓말 탐지기 지식, 현재 자동 상호작용과 계약의 라운드 상태를 정리한다.
- 수정 구슬·보위 나이프 등 부모 카드 효과에서 들어온 경우 부모 효과와 남은 차례 재개를 취소한다.
- 양측 손패를 각자의 버린 더미로 옮기고 스탠드를 해제한 뒤 기존 `StartRound`를 정확히 한 번 호출한다.
- 새 라운드 최초 공개·비공개 배분은 기존 예외대로 자동 발동하지 않는다.
- 거절은 부활초 원본만 버리고 영혼·라운드 번호·다른 손패를 바꾸지 않는다.
- 플레이어 화면 입력, 진행 세션, 보상 등록과 적 자동 선택 정책은 AC-06에 남긴다.

## 7. 플레이어 입력과 적 AI

### 7.1 플레이어 입력

- 자동 효과 보류 중에는 전용 안내와 현재 선택지만 표시한다.
- 입력에는 증가형 상호작용 ID와 옵션 ID를 함께 사용해 오래된 클릭과 중복 제출을 거절한다.
- 적 차례 중 플레이어가 화염 방사기 대상으로 선택해야 하는 경우에도 전투를 일시 중단하고 플레이어 입력을 기다린다.
- 표시 모델은 규칙을 다시 계산하지 않고 현재 보류 상호작용의 안전한 읽기 모델만 사용한다.

### 7.2 적 AI

- 적이 자동 카드 소유자 또는 선택 대상이면 정상 행동 정책과 분리된 자동 효과 선택 정책을 사용한다.
- 동일한 공개 상태와 시드는 동일한 선택을 만들어야 한다.
- 독극물은 생존 가능성과 스탠드 가능 여부를 비교한다.
- 부활초는 불리한 공개 상태이면서 양측 영혼 조건을 만족할 때 우선한다.
- 거짓말 탐지기는 공개 추론 분포를 가장 비슷하게 둘로 나누는 선언 값을 우선한다.
- 화염 방사기와 회중시계는 공개 카드와 사용 상태만으로 선택한다.
- 거짓말 탐지기의 판정 외에는 플레이어 비공개 숫자를 직접 전달하지 않는다.

## 8. 런·보상 연결

- 자동 카드 5종은 안정적인 정의 키와 물리 카드 ID를 사용한다.
- 시작 덱에는 자동 카드를 추가하지 않는다.
- 개별 카드 효과와 UI 검증이 끝나기 전에는 기본 보상 풀에 등록하지 않는다.
- AC-06에서 5종을 일반 보상 풀에 추가한다.
- 카드 등급이 확정되지 않았으므로 높은 등급 보상 풀에는 자동으로 추가하지 않는다.
- 사기꾼 프로필의 거짓말 탐지기 등 적 덱 변경도 해당 카드의 양측 효과와 정보 은닉 검증 뒤에만 적용한다.
- 런 재시작 시 획득한 자동 카드와 전투 중 비교 지식·회복 예약·보류 상호작용은 초기화한다.

## 10. 완료 조건

다음을 모두 만족하면 자동 발동 카드 시스템 1차 범위를 완료한다.

- 최초 배분 예외와 모든 공개 유입 경로가 자동 테스트로 고정된다.
- 플레이어·적 양측에서 5종 효과와 선택이 동일한 규칙으로 실행된다.
- 자동 효과 뒤 버스트와 원래 행동 재개가 정확히 한 번 발생한다.
- 부활초 초기화가 피해·독극물 회복·외부 효과를 중복 처리하지 않는다.
- 거짓말 탐지기가 실제 비공개 값을 공용 결과·UI·AI 관측에 노출하지 않는다.
- 카드 총수, 물리 ID, 버린 더미와 사용 상태가 반복 전투에서 보존된다.
- 독립 전투와 런 전투 UI에서 선택·결과·재시작을 확인한다.
- 대상 테스트, 전체 EditMode, 두 씬, 두 해상도와 Console 검증 결과를 기록한다.
- 외부 에셋·오픈소스 추가가 없음을 기록한다.
