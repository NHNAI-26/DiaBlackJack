# 1. 게임 개요

플레이어는 악마에게 납치되어 지옥의 블랙잭 대회에 참가한다.

각 스테이지에서는 한 명의 적과 여러 라운드에 걸쳐 블랙잭 대결을 진행한다. 이곳에서는 단순히 21에 가까운 숫자를 만드는 것뿐 아니라 총기, 흉기, 독극물, 마법과 악마의 힘을 사용해 상대를 직접 버스트시키는 행위도 허용된다.

플레이어는 12개의 영혼을 가지고 게임을 시작한다. 플레이어의 영혼은 런 전체에서 유지되며, 일반적으로 적 한 명을 쓰러뜨려도 자동으로 회복되지 않는다.

각 적은 해당 스테이지에서만 사용하는 영혼을 가진다.

- 일반 적: 영혼 2~3
- 강한 일반 적: 영혼 3~4
- 엘리트: 영혼 4~5
- 보스: 영혼 6~8

플레이어 또는 적의 영혼이 0이 될 때까지 블랙잭 라운드를 반복한다.

# 2. 핵심 게임 경험

## 2.1 블랙잭을 이용한 전투

카드 숫자의 합을 21에 가깝게 만드는 기본적인 블랙잭 규칙을 사용한다.

그러나 각 숫자 카드에는 상대의 카드를 제거하거나, 상대를 강제로 히트시키거나, 비공개 카드를 추측하는 전투 효과가 존재한다.

따라서 플레이어는 다음 두 목적 사이에서 판단해야 한다.

- 좋은 숫자 합계를 만든다.
- 카드를 사용해 상대의 상태를 무너뜨린다.

강력한 효과를 가진 카드가 반드시 좋은 숫자를 가진 것은 아니며, 효과를 사용하기 위해 비공개 카드를 공개하는 것도 가능하다.

## 2.2 영혼을 체력과 자원으로 동시에 사용

영혼은 플레이어의 체력인 동시에 악마와 계약하거나 상점을 이용하는 자원이다.

영혼을 사용하면 즉시 강해질 수 있지만, 이후 전투에서 패배할 가능성이 높아진다.

플레이어는 다음과 같은 선택을 반복한다.

- 안전하게 영혼을 보존한다.
- 영혼을 소비해 전투를 빠르게 끝낸다.
- 낮은 영혼 상태를 활용하는 발악 빌드를 구성한다.
- 악마 계약을 반복해 강력한 효과를 얻는다.

## 2.3 악마 계약

플레이어는 자신의 차례에 영혼을 소비해 악마와 계약할 수 있다.

악마는 일반 카드보다 훨씬 강력한 효과를 제공하지만, 반드시 플레이어에게 불리한 대가를 요구한다.

악마 계약은 전투의 규칙을 일시적으로 변경하는 고위험·고효율 행동이다.

계약 대상의 제공 방식과 악마 덱에서 악마를 선택하는 세부 절차는 현재 확정하지 않는다. 프로토타입에서는 구현 난도에 따라 임시 방식을 사용하고 이후 변경할 수 있다.

## 2.4 상대의 추론을 역이용하는 심리전

적 AI는 플레이어의 실제 비공개 카드나 덱의 다음 카드를 직접 알 수 없다.

AI는 다음 정보만을 사용한다.

- 플레이어의 공개 카드
- 이미 사용되거나 버려진 카드
- 현재까지 공개된 카드 숫자
- 플레이어가 사용한 행동
- 활성화된 카드 효과와 악마 계약
- 거짓말 탐지기 등으로 얻은 정보
- 플레이어 덱의 공개된 구성 정보

플레이어는 AI가 현재 어떤 숫자를 의심하고 있는지 일부 확인할 수 있다.

예시:

- 숫자 7일 가능성: 30%
- 숫자 8일 가능성: 25%
- 숫자 5일 가능성: 15%
- 기타 숫자: 30%

AI의 정확한 다음 행동을 항상 공개하지는 않는다.

일부 적은 공격적인 성향, 추측 중심 성향, 안전한 스탠드 성향 등 고유한 사고방식을 가진다.

# 3. 전체 게임 구조

## 3.1 런 시작

플레이어는 다음 상태로 게임을 시작한다.

- 영혼 12
- 기본 일반 덱
- 기본 악마 계약 목록 또는 악마 덱
- 기본 유물 또는 캐릭터 특성
- 승점 및 전투 상태 없음

## 3.2 스테이지 진행

플레이어는 지도 없이 `전투 → 이벤트 → 전투 → 이벤트 → 보스` 순서의 런을 진행한다.

스테이지 종류는 다음과 같다.

- 상대 선택 전투
- 보스 전투
- 상점
- 휴식
- 무작위 사건

일반 전투에 진입하기 전 서로 다른 상대 후보 2명을 제시하고 플레이어가 한 명을 선택한다. 후보 조합은 일반 적 2명 또는 일반 적 1명과 엘리트 1명으로 구성하며, 엘리트 2명을 동시에 제시하지 않는다. 선택한 적과 데블랙잭 게임을 시작한다.

이벤트 스테이지에는 휴식, 상점, 무작위 사건 중 하나가 무작위로 등장한다. 이벤트 종류별 등장 가중치는 플레이 테스트 후 확정한다.

## 3.3 전투 종료

다음 중 하나가 발생하면 전투가 종료된다.

- 적의 영혼이 0이 된다.
- 플레이어의 영혼이 0이 된다.
- 특정 보스의 고유 패배 조건이 충족된다.

## 3.4 런 종료

플레이어의 영혼이 0이 되면 런이 종료된다.

최종 보스를 쓰러뜨리면 런에서 승리한다.

# 4. 용어 구조

게임의 진행 단위는 다음과 같이 구분한다.

## 런

게임을 시작해 최종 보스를 쓰러뜨리거나 플레이어가 패배할 때까지의 전체 과정이다.

## 스테이지

상대 선택을 포함한 하나의 전투 또는 휴식·상점·무작위 사건 중 하나의 이벤트를 의미한다.

## 라운드

각자 두 장의 카드를 받은 시점부터 한쪽이 버스트하거나 양쪽이 스탠드할 때까지의 과정이다.

한 스테이지에서는 여러 라운드가 진행될 수 있다.

## 차례

라운드 안에서 플레이어나 적이 행동 하나를 선택하는 단위다.

# 5. 영혼 시스템

## 5.1 플레이어 영혼

플레이어는 영혼 12개로 런을 시작한다.

플레이어 영혼은 스테이지가 끝나도 유지된다.

전투에서 승리하더라도 기본적으로 영혼은 회복되지 않는다.

영혼이 0이 되면 런에서 패배한다.

## 5.2 적 영혼

적 영혼은 현재 스테이지에서만 사용된다.

적을 쓰러뜨리면 남은 영혼과 관계없이 해당 적의 영혼은 소멸한다.

적 영혼의 권장 범위는 다음과 같다.

| 적 등급 | 권장 영혼 |
| --- | --- |
| 일반 적 | 2~3 |
| 강한 일반 적 | 3~4 |
| 엘리트 | 4~5 |
| 보스 | 6~8 |

## 5.3 라운드 피해량

현재 프로토타입의 임시 피해량은 다음과 같다.

| 결과 | 플레이어 피해 | 적 피해 |
| --- | --- | --- |
| 플레이어 폴드 | 영혼 1 | 없음 |
| 플레이어 라운드 패배 | 영혼 2 | 없음 |
| 적 라운드 패배 | 없음 | 영혼 1 |
| 정확히 21로 적에게 승리 | 없음 | 영혼 2 |
| 카드 특수 효과 | 카드별 적용 | 카드별 적용 |

플레이어와 적의 피해량은 비대칭으로 설계한다.

플레이어의 영혼은 런 전체에서 유지되지만, 적의 영혼은 하나의 스테이지에서만 사용되기 때문이다.

피해량은 플레이 테스트 결과에 따라 변경할 수 있다.

# 6. 덱 구조

## 6.1 기본 덱 크기

프로토타입의 플레이어 시작 덱은 16~20장으로 구성한다.

초기 권장 구성은 다음과 같다.

- 숫자 1~10 카드 각 2장
- 총 20장

또는 플레이어 캐릭터에 따라 일부 숫자 카드의 수와 효과를 변경할 수 있다.

## 6.2 덱 빌딩 대상

플레이어는 두 종류의 덱 또는 카드 집합을 성장시킨다.

### 일반 덱

실제 블랙잭 라운드에서 뽑는 숫자 카드 덱이다.

일반 덱에서는 다음 행동이 가능하다.

- 새로운 카드 추가
- 기존 카드 제거
- 카드 강화
- 동일 숫자의 다른 효과 카드로 교체
- 특정 숫자 카드의 수량 조정

### 악마 덱 또는 계약 목록

계약 행동에서 등장할 악마를 관리한다.

다음과 같은 성장이 가능하다.

- 새로운 악마 추가
- 악마 제거
- 악마 강화
- 악마의 대가 완화
- 특정 악마의 등장 확률 증가
- 계약 비용 조절

계약 대상을 어떤 방식으로 제시할지는 현재 미확정이다.

## 6.3 전투 중 덱 순환

전투가 시작될 때 플레이어와 적은 자신의 덱을 섞는다.

각 라운드가 종료되어도 덱을 전부 초기화하지 않는다.

사용된 카드는 버린 카드 더미에 놓는다.

다음 라운드에는 남아 있는 덱에서 카드를 뽑는다.

덱이 모두 소진되면 버린 카드 더미를 섞어 새로운 덱으로 사용한다.

전투가 종료되면 덱과 버린 카드 더미를 초기 상태로 복구한다.

이 구조를 통해 다음 전략이 가능하다.

- 남은 높은 숫자 카드의 수를 계산한다.
- 자동 권총이 남아 있을 확률을 추측한다.
- 에이스가 이미 사용되었는지 확인한다.
- 다음 라운드를 위해 특정 카드를 보존한다.
- 덱 순환 속도를 높여 원하는 카드에 다시 접근한다.

# 7. 라운드 규칙

## 7.1 라운드 시작

라운드가 시작되면 플레이어와 적은 각각 자신의 덱에서 카드 2장을 뽑는다.

각자는 두 장을 모두 확인한 뒤 다음과 같이 배치한다.

- 카드 1장: 공개 카드
- 카드 1장: 비공개 카드

적의 비공개 카드는 플레이어에게 보이지 않는다.

플레이어의 비공개 카드는 적 AI가 직접 확인할 수 없다.

## 7.2 차례 행동

자신의 차례에는 다음 행동 중 하나를 선택한다.

### 히트

덱 맨 위에서 카드 1장을 뽑아 자신의 공개 카드에 추가한다.

### 스탠드

현재 라운드에 더 이상 차례를 진행하지 않는다.

상대 카드 효과에 의해 스탠드가 취소될 수 있다.

### 폴드

영혼 1개를 잃고 현재 라운드를 포기한다.

현재 라운드에서 사용한 카드는 버린 카드 더미로 이동한다.

남아 있는 덱 상태를 유지한 채 새로운 라운드를 시작한다.

### 카드 사용

사용 가능한 공개 카드 한 장의 효과를 발동한다.

비공개 카드도 공개하면서 사용할 수 있다.

### 체인지

자신의 비공개 카드를 버리고 일반 덱 맨 위 카드 2장 자신만 확인한다.

그 중 1장을 새로운 비공개 카드로 받고, 나머지는 버린다.

라운드당 1번만 할 수 있다.

### 계약

영혼 1개를 버린다. 내 악마 덱에 있는 악마 카드 3장을 확인한다. 그 중 하나와 계약하고, 나머지는 버린다.

악마는 강력한 능력과 그에 상응하는 대가를 제공한다.

계약 효과는 해당 스테이지가 종료될 때까지 적용된다.

## 7.3 자동 발동 카드

자동 발동 카드가 공개 카드에 추가되면 즉시 효과를 처리한다.

**라운드 시작 시 최초 카드 분배에서는 자동 발동이 적용되지 않는다.**

효과 처리 후 카드의 위치는 카드별 규칙을 따른다.

## 7.4 버스트

다음과 같은 상황에서 버스트될 수 있다.

- 공개 카드 합이 21을 초과한다.
- 라운드 종료 시 카드의 최종 합이 21을 초과한다.
- 카드 효과가 대상을 버스트시킨다.

본 게임에서 `[버스트]`는 단순한 숫자 초과만을 의미하지 않는다.

카드나 악마 효과로 해당 라운드에서 탈락하는 모든 결과를 통합해 버스트라고 부른다.

버스트한 대상은 즉시 해당 라운드에서 패배한다.

# 8. 라운드 종료

다음 중 하나가 발생하면 라운드가 종료된다.

- 플레이어가 버스트한다.
- 적이 버스트한다.
- 플레이어가 폴드한다.
- 양쪽 모두 스탠드한다.

## 8.1 버스트 또는 폴드

한쪽이 버스트하거나 폴드하면 즉시 승패를 처리한다.

- 플레이어 버스트: 플레이어 영혼 2 감소
- 플레이어 폴드: 플레이어 영혼 1 감소
- 적 버스트: 적 영혼 1 감소
- 카드 효과에 별도의 영혼 피해가 있으면 추가 적용

## 8.2 양쪽 스탠드

양쪽 모두 비공개 카드를 공개한다.

각자의 카드 숫자 합을 계산한다.

에이스는 1 또는 11 중 합계가 21 이하에서 가장 커지는 값을 자동으로 선택한다.

다음 순서로 승자를 결정한다.

1. 21을 넘은 대상은 일반 승리 대상에서 제외한다.
2. 21 이하에서 21에 더 가까운 대상이 승리한다.
3. 양쪽 모두 21을 넘었다면 21에 더 가까운 대상이 승리한다.
4. 무승부일 경우 일반적으로 플레이어의 승리로 간주한다.

## 8.3 정확히 21

플레이어가 정확히 21로 라운드에서 승리하면 적에게 영혼 피해 2를 준다.

# 9. 전투 종료와 회복

## 9.1 승리

적의 영혼이 0이 되면 해당 스테이지에서 승리한다.

전투에서 승리해도 플레이어 영혼은 자동으로 회복되지 않는다.

플레이어는 카드 보상, 상점, 휴식 또는 사건을 통해 회복 수단을 확보해야 한다.

## 9.2 패배

플레이어의 영혼이 0이 되면 런이 종료된다.

## 9.3 선택적 회복 수단

자동 회복은 제공하지 않지만 다음과 같은 선택형 회복 수단을 제공한다.

- 휴식 노드에서 영혼 회복
- 상점에서 영혼 회복 구매
- 특정 유물의 조건부 회복
- 특정 카드 또는 악마 효과를 통한 회복
- 사건을 통한 대가성 회복

회복은 항상 다른 성장 선택지를 포기하거나 자원을 소비하도록 설계한다.

# 14. 적 AI 설계

## 14.1 AI 공정성 원칙

적 AI는 게임 내부적으로 존재하는 비공개 정보를 직접 사용하지 않는다.

AI 판단 함수에는 플레이어 비공개 카드의 실제 숫자와 덱의 실제 다음 카드가 전달되지 않아야 한다.

AI는 공개 정보와 확률만으로 판단한다.

## 14.2 AI 추론 정보

AI는 다음과 같은 추론 데이터를 관리한다.

- 각 숫자 카드가 플레이어 비공개 카드일 확률
- 플레이어 덱에 남은 카드의 예상 분포
- 플레이어가 버스트할 확률
- 플레이어가 특정 카드 효과를 보유할 확률
- 플레이어가 계약을 선택할 가능성
- 플레이어의 반복 행동 성향

## 14.3 플레이어에게 제공되는 정보

플레이어에게 AI의 전체 내부 상태를 공개하지 않는다.

대신 일부 추론 결과를 UI로 표시한다.

예시:

- `상대는 높은 숫자를 의심하고 있습니다.`
- `상대가 가장 의심하는 숫자: 7, 8`
- `상대의 확신도: 낮음`
- `상대는 당신이 버스트에 가깝다고 판단합니다.`

상세 모드에서는 숫자별 추정 확률을 표시할 수 있다.

적의 다음 행동 자체는 항상 공개하지 않는다.

특정 적이나 보스만 행동을 사전에 예고하도록 설계할 수 있다.

# 16. 보상과 성장

## 16.1 전투 보상

적을 쓰러뜨리면 카드 보상 3장을 제시한다. 플레이어는 카드 1장을 현재 런 덱에 추가하거나, 아무것도 선택하지 않고 보상을 건너뛸 수 있다.

일반 적은 일반 카드 보상 풀에서 3장을 제시한다. 엘리트 적은 제시하는 3장 모두 높은 등급의 카드로 확정한다. 높은 등급의 기준은 카드 등급 체계를 확정할 때 정한다.

카드 제거·강화, 영혼 회복과 유물 획득은 전투 보상과 분리하며 휴식, 상점 또는 무작위 사건에서 제공한다.

## 16.2 카드 강화

카드 강화는 숫자를 바꾸기보다 효과를 강화하는 방향을 우선한다.

예시:

- 수정 구슬: 확인 카드 2장 → 3장
- 자동 권총: 숫자 1개 선언 → 조건부로 2개 선언
- 군용 나이프: 강제 히트 후 카드를 버릴 수 없음
- 진정제: 사용 시 부가 효과 획득
- 수면 가스: 사용 후 덱 정보 확인

구체적인 강화안은 프로토타입 이후 확정한다.

## 16.3 유물

유물은 덱에 들어가지 않는 지속 효과다.

예시:

- 전투당 첫 계약 비용 감소
- 최초 폴드 시 영혼 손실 무효
- 정확히 21로 승리하면 영혼 회복
- 자동 권총 사용 시 추측 확률 표시 강화
- 덱을 다시 섞을 때 카드 1장 확인
- 휴식 회복량 증가

## 17.2 AI 추론 패널

AI가 플레이어를 어떻게 추측하고 있는지 간략하게 표시한다.

표시 방식은 적의 특성과 게임 난도에 따라 조정한다.

- 쉬운 난도: 숫자별 확률 표시
- 보통 난도: 가장 의심하는 숫자 2~3개 표시
- 어려운 난도: 추론 방향과 확신도만 표시
- 특정 보스: 잘못된 추론 정보를 표시할 가능성 존재

잘못된 정보를 제공하는 보스가 존재할 경우, 해당 능력은 사전에 명확하게 설명한다.

## 17.3 사탄 계약 UI

사탄과 계약하면 화면에 남은 차례를 크게 표시한다.

예시:

`사탄의 계약 종료까지 4턴`

각 정상 차례가 끝날 때 숫자가 감소한다.

마지막 차례에는 화면과 사운드를 통해 버스트 위험을 강하게 경고한다.

# 22. 현재 보류된 사항

다음 항목은 아직 확정하지 않는다.

## 계약 대상 제공 방식

- 무작위 악마 여러 명 중 선택
- 미리 구성한 악마 덱에서 드로우
- 전투 전에 장착한 악마만 사용
- 계약할 때마다 악마 후보가 누적

프로토타입에서는 가장 빠르게 구현 가능한 방식을 임시로 사용한다.

## 일반 카드 세부 문구

수면 가스, 위협용 해머 등 일부 카드는 1대1 환경에서 문구가 모호할 수 있다.

현재 효과의 의도는 유지하되, 실제 구현 후 UI에 맞게 표현을 수정한다.

## 악마 카드 숫자의 합계 포함 여부

악마 카드가 공개 카드에 추가될 때 숫자를 블랙잭 합계에 포함할지 추후 결정한다.

## 자동 발동 카드 처리

효과 처리 후 카드를 버릴지, 공개 카드로 유지할지 카드별로 정해야 한다.

## 오망성 분배

바포메트 계약 시 오망성을 플레이어와 적의 덱에 몇 장씩 넣을지 정해야 한다.

## 적 행동 정보 공개 범위

AI가 추측하는 숫자 확률은 표시하지만, 정확한 다음 행동을 어느 정도까지 표시할지는 난도와 적 유형에 따라 조정한다.

# 24. 핵심 차별점

1. 블랙잭의 숫자 계산과 카드 전투 효과가 동시에 작동한다.
2. 영혼이 체력과 계약 비용으로 함께 사용된다.
3. 적 AI가 비공개 정보를 부정하게 확인하지 않고 확률적으로 추론한다.
4. 플레이어는 AI가 자신을 어떻게 추측하고 있는지 일부 확인할 수 있다.
5. 악마 계약은 강력한 효과와 명확한 자멸 위험을 동시에 제공한다.
6. 전투마다 적의 영혼과 덱, 행동 성향이 달라진다.
7. 플레이어의 영혼과 덱 상태가 런 전체에 누적된다.
