## 4. 착수 전 확정 사항

### 4.1 작업 경계

- 코어 루프 완료 기록은 수정 없이 유지한다.
- 진행 시스템은 전투 결과와 플레이어 지속 상태만 다룬다.
- 최초 구현은 일반 전투와 최종 보스 전투만 포함한다.
- 보상, 휴식과 분기 지도는 별도 후속 작업으로 둔다.

### 4.2 임시 검증 경로

`일반 전투 1 → 일반 전투 2 → 최종 보스 전투`

이 경로는 진행 구조 테스트용이며 최종 콘텐츠 수량이 아니다. 이천서의 SP-00 진행 지시에 따라 최초 구현 기준으로 확정했다.
